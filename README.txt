

 This is one type of python based CHATBOT(vennichat) project. This venni chat is useful for Hotels, 
 Institutions, Colleges etc. Hotels have halls for birthday parties, marriages, etc so that customers 
 visits their websites that time this vennichat helps to take their details and stored. Hotel executives 
 will call back them to increase their business.if customer likes your websites they will give their 
 contact details so that it is included with NLP domain also. In this code we used Tkinter module and some
 other libraries.
